# Andriod_studio
Android studio tool을 이용해 APP을 개발해본다.
- Android 앱개발을 위한 공식통합개발환경(IDE)
- Intellij IDEA기반 (pycharm과 동일)
- 장점
    - 유연한 gradle 기반 빌드시스템
    - 에뮬레이터 : 핸드폰을 pc에서 구동가능함
    - 모든 android기기 지원
    - C++ 및 NDK지원
    - 빠른 빌드
